# AtlasLoot Enhanced - Ebonhold Tomes Edition

> [!NOTE]
> This version adds Ebonhold Tomes to the World Events section.

## Overview
AtlasLoot Enhanced for World of Warcraft 3.3.5 is a comprehensive loot browser. It allows players to browse loot tables for dungeons, raids, factions, and events without waiting for drops.

## Features
- **Loot Tables**: View loot for every dungeon (Normal/Heroic) and raid in WotLK, TBC, and Classic.
- **Collections**: Browse armor sets (Tier, PvP, Crafted), mounts, pets, and tabards.
- **Wishlist**: Alt-click items to add them to a personal wishlist.
- **Search**: Powerful search filter to find items by name or source.
- **Ebonhold Integration**: Specifically updated to include custom Ebonhold Tomes in the World Events category.

## Usage
- Open the browser: `/al` or `/atlasloot`
- Options menu: `/atlasloot options`
- Toggle minimap button: `/atlasloot mmb`
